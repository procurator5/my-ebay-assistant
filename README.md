# my-ebay-assistant

## Install

This software uses python3. I also use framework Django and database PostreSQL.
You need install some packages:

* python 3.5+
* PostgreSQL 9.1+
* django 1.11+
* mttp
* django-cron
* django-admin-tools

## Configure

My-ebay-assistant uses eBay API. You must registered application on the ebay.com.
